# ctx-init

Bootstrap a tool-agnostic AI context system for your project

## How to

There is a Docker-based script [`scripts/ci.sh`](./scripts/ci.sh) for CI and local container-based runs.

```bash
# Run the CLI (default mode)
./scripts/ci.sh

# Or explicitly
./scripts/ci.sh run

# Run tests
./scripts/ci.sh test
```
